Before You Start

Before installing any of our mods, there are a few things you should prepare to ensure a smooth installation process:

1

Disable Your Antivirus

Game mods may be flagged by antivirus software as potentially unwanted programs. This is a false positive. Temporarily disable your antivirus during installation, or add an exception for our mods.

2

Close Your Game

Make sure the game you're modding is completely closed before installing any mods. Running the installer while the game is active can cause conflicts.

3

Run as Administrator

Always run our installers with administrator privileges to ensure they have the necessary permissions to modify game files.

Important Note

Using mods in online games carries a risk of being banned. We design our mods to be as undetectable as possible, but we cannot guarantee 100% safety. Use at your own risk and follow our guidelines for the safest experience.

Downloading Your Mod

Follow these steps to download your chosen mod from our website:

1

Browse Our Catalog

Visit our Game Mods page and find the game you want to mod. Click on the game card to view available mods.

2

Select Your Mod

On the game details page, you'll find a list of available mods. Click the "Download" button next to the mod you want.

3

Save the File

Choose a location on your computer to save the mod file. We recommend creating a dedicated folder for your game mods to keep them organized.

Installation Process

Once you've downloaded the mod file, follow these steps to install it:

1

Extract the Files

Right-click the downloaded ZIP file and select "Extract All" or use your preferred extraction tool.

2

Run the Installer

Navigate to the extracted folder and right-click on the installer file (usually named "Setup.exe" or "Install.exe"). Select "Run as administrator".

3

Follow the Installation Wizard

The installer will guide you through the installation process. In most cases, you'll need to:

  • Accept the terms of service
  • Select your game installation directory (if not detected automatically)
  • Choose which mod features to install
  • Wait for the installation to complete
4

Complete Installation

Once the installation is complete, you may be prompted to restart your computer. It's recommended to do so to ensure all changes take effect properly.

Launching the Mod

After installation, you'll need to launch the mod correctly to use it with your game:

1

Launch the Mod Loader

Find the mod loader shortcut on your desktop or in the installation folder. Right-click and select "Run as administrator".

2

Configure Settings

In the mod loader interface, you can configure various settings for the mod, such as enabling/disabling specific features or adjusting parameters.

3

Launch the Game

Click the "Launch Game" button in the mod loader, or launch the game normally if the mod loader doesn't have this option. The mod will automatically inject into the game.

4

Access Mod Menu In-Game

Once in-game, you can access the mod menu by pressing the designated hotkey (usually Insert, Home, or F11). This will bring up the mod interface where you can toggle features while playing.

Default Hotkeys

The default hotkeys for our mods are:

  • Insert or Home: Open/close the mod menu
  • Arrow Keys: Navigate the menu
  • Enter: Select an option
  • Backspace: Go back
  • End: Panic key (instantly closes the mod)

These can be customized in the mod loader settings.

Troubleshooting

If you encounter issues with our mods, try these common troubleshooting steps:

1

Mod Not Injecting

Solution: Make sure you're running both the mod loader and the game as administrator. Also, check if your antivirus is blocking the mod. Add an exception or temporarily disable it.

2

Game Crashes on Launch

Solution: This often happens when the mod is incompatible with your game version. Check if there's an update available for the mod. If not, try reinstalling both the game and the mod.

3

Features Not Working

Solution: Some features may be disabled by default for safety. Check the mod menu to ensure they're enabled. Also, some features might only work in specific game modes or scenarios.

4

Missing DLL Errors

Solution: Install the required Visual C++ Redistributables. Links to these are provided in the mod's readme file or on our website.

Need More Help?

If you're still experiencing issues, please contact our support team. Be sure to include detailed information about the problem, your system specifications, and any error messages you're receiving.

Updating Your Mods

Keeping your mods updated is crucial for compatibility with the latest game versions and to access new features:

1

Check for Updates

Our mod loader will automatically check for updates when launched. You can also manually check by clicking the "Check for Updates" button in the mod loader interface.

2

Download the Update

If an update is available, the mod loader will prompt you to download it. Click "Download" to proceed.

3

Install the Update

Once downloaded, the mod loader will automatically install the update. You may need to restart the mod loader and/or your game for the changes to take effect.

4

Verify the Update

After installation, check the version number in the mod loader to confirm that the update was successful. The version number should match the latest version shown on our website.

Stay Informed

Subscribe to our email notifications on the Contact page to receive alerts when new updates are released for your favorite mods.
